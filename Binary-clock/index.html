Binary Clock

Welcome to my Binary Clock project. Below, you'll find details about the clock, its core functionalities, and the design approach.

How It Was Built

The Binary Clock is built using the C programming language and a Raspberry Pi equipped with an LED Sense HAT. The clock represents time in binary format, utilizing the LED matrix to display hours, minutes, and seconds in binary. The LED Sense HAT provides a versatile way to visually represent time through a grid of LEDs that light up based on the current time's binary representation.

Core Functionalities

  • Binary Representation: Displays time in binary format with LEDs corresponding to each bit.
  • Three Columns: Shows hours, minutes, and seconds, each split into two 4-bit groups.
  • Real-Time Updates: Continuously updates the LED display to reflect the current time.

Design Approach

The Binary Clock leverages the capabilities of the Raspberry Pi and the LED Sense HAT to provide a real-time, binary-based time display. The C programming language is used to handle the logic for converting and displaying time in binary format. The design focuses on using the LED matrix effectively to ensure clear and accurate representation of time.

Challenges Faced

  • Implementing real-time updates for the binary representation of time on the LED matrix.
  • Designing a method to convert and display time accurately in binary format using the LED Sense HAT.
  • Managing the hardware and software integration between the Raspberry Pi and the LED Sense HAT to ensure smooth functionality.
